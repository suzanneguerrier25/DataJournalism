Bird Calls: Injured Birds in NYC 2022

Context:

I used data from NYC's Urban Park Rangers which contains all calls made about animals, for example, finding an injured stray cat, and edited it to focus on calls about birds- injured, unhealthy, or others. I'm using this data to determine where these calls are happening, by borough, and by a more precise location in the city, as well as what types of birds are being injured.

Data Sources:

Urban Park Ranger Animal Condition Response

Provided by the Department of Parks and Recreation (DPR), Accessed through NYC OpenData

Data Cleansing Pipeline:

  1. Downloaded the data as a csv file and import to google sheets
  2. Used google sheets to edit the data, deleting columns that weren't necessary to my narrative (ex. if the police was called) and removing rows that contained non-birds / non-raptors (eg. hawk)
  3. Downloaded the edited csv file to VS Code, and used a format_data.py file in conjunction with the json library to make a json file with the organized data
  4. Organized the data as a dictionary with the time and date as the key- separating the indivdual calls, and a list with the other attributes as the values
  5. I also removed any extra space as well as edited the Birds, Domestic to be Domestic Birds to facilitate conversion. I also edited any addresses with quotations for the same purpose
